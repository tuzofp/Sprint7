# Sprint7

Proyecto Sprint 7: Análisis Exploratorio de Datos con Streamlit Descripción del proyecto
Esta aplicación web fue desarrollada como parte del Sprint 7 del bootcamp de Ciencia de Datos. Su propósito es facilitar el análisis exploratorio de datos mediante una interfaz interactiva construida con Streamlit, utilizando Pandas para manipulación de datos y Plotly para visualizaciones dinámicas.
El proyecto está diseñado para permitir a cualquier usuario:

Visualizar los datos en formato tabular.
Seleccionar columnas para generar gráficos interactivos.

⚙ Funcionalidades principales
- st.file_uploader: permite cargar archivos CSV.
- st.dataframe: muestra una vista previa de los datos.
- st.selectbox: permite elegir columnas para graficar.
- plotly.express: genera gráficos interactivos en tiempo real.
- st.plotly_chart: integra los gráficos en la app web
